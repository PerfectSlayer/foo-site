Event

2016/12/13 - Functional Event Sourcing: une approche simple et directe

CQRS / Event Sourcing / DDD

Détails

19h15 à 19h30 : Accueil

19h30 - 20h30 : CQRS / Event Sourcing / DDD

En stockant l’histoire du système plutôt que son état actuel, l’Event Sourcing est un technique radicale pour implémenter Domain Driven Design efficacement. Mais beaucoup s’arrêtent avant d’avoir commencé, convaincu qu’une telle solution sera difficile à mettre en œuvre. Un exemple court en F# - mais posant les bonnes questions - montre comment implémenter Event Sourcing de façon simple, claire et production ready, en laissant la plus grosse part à un code métier devenu limpide ! Aucune connaissance préalable du F# n’est requise…
